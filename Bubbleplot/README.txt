###Summary###

The function of this code is to input the number of antennas and the corresponding relative abundance of each organ, and output the bubble map and the composite map of the abundance map.


###Depedencies###
R version 4.4.1 is required.
Before using it, packages "dplyr", "ggplot2", "readxl", "purrr" , "patchwork", "tidyr"need to be installed.


###Protocol Steps###
# Set the working directory to the folder named "Bubbleplot" in R environment (or RStudio).
# Select all codes and click "run", the figure will be generated automatically.

# After running the program, you will get: bubble_plot_with_stacked_bar.svg.
