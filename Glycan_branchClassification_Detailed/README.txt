The function of this program is to classify and count the different glycan structures.


Double-click "Glycan_Classification_Detailed.exe"to get three files:
1) linkages_output.txt, which includes 3 glycan types, 4 core types, 14 branch types and antenna count.
2) linkages_stats_output.txt, which includes 14 branch types.
3) output.txt, which includes 3 glycan types.
